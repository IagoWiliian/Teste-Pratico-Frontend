🐾 Petshop  — Frontend
Aplicação desenvolvida em React  com consumo de uma API em Spring Boot para comparar preços entre diferentes petshops e encontrar a melhor opção para o cliente.


🚀 Tecnologias utilizadas
React

Vite

Axios

TypeScript

CSS
